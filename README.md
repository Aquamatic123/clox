# clox
A lightweight, bytecode virtual machine implementation of the Lox programming language, written in C. This project is based on the clox interpreter from Robert Nystrom's Crafting Interpreters.

# Installation
To install you will need make and gcc. You will then just need to clone the repo, make, and run 
```
git clone https://github.com/Aquamatic123/clox.git
cd clox
make
./clox
```
